# Eden Androids

Eden Androids é uma plataforma fictícia de e-commerce especializada na venda de androides para uso doméstico, profissional e empresarial.

## Objetivo do Projeto

O projeto foi criado para estudo prático e portfólio, com foco em desenvolvimento front-end usando React. A proposta é evoluir gradualmente conceitos como componentização, rotas, layout responsivo, catálogo de produtos, carrinho fictício e, futuramente, integração com back-end.

## Funcionalidades Planejadas por MVP

### MVP 1 — Front-end

- Home
- Catálogo
- Página de detalhes
- Filtros
- Carrinho fictício
- Layout responsivo

### MVP 2 — Dados

- Produtos vindos de JSON ou API
- Carrinho persistindo no localStorage
- Busca e filtros melhores

### MVP 3 — Back-end

- Node.js/Express ou Java Spring
- PostgreSQL
- Cadastro/login
- Pedidos
- Painel administrativo

### MVP 4 — Funcionalidades avançadas

- Dashboard administrativo
- Upload de imagem
- Status de pedido
- Avaliações
- Recomendação de produto
- Simulação de financiamento/assinatura
- Chatbot de atendimento

## Como Rodar o Projeto Localmente

```bash
git clone https://github.com/FelipeJordaoFernandes/eden-androids.git
cd eden-androids
npm install
npm run dev
```

Depois de iniciar o servidor de desenvolvimento, acesse a URL exibida no terminal.
